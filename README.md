# DASI-Skript

## Überblick
Dieses Skript ermöglicht die einfache Sicherung und Wiederherstellung von Windows-Benutzerprofilen sowie Daten verschiedener Browser, E-Mail-Programme und WLAN-Profile. Es bietet eine interaktive Auswahl per Eingabeaufforderung.

## Funktionen
- **Windows-Benutzerprofil sichern und wiederherstellen**
- **Browser-Daten sichern und wiederherstellen:**
  - Firefox
  - Edge
  - Chrome
  - Brave
- **E-Mail-Client Thunderbird sichern und wiederherstellen**
- **Liste installierter Programme exportieren und wiederherstellen**
- **Export und Import von WLAN-Profilen**

## Installation & Nutzung
1. **Programm starten**  
   Führe die Programmdatei mit Administratorrechten aus:
   
   Wähle dann die gewünschte Option per Eingabeaufforderung.

## Voraussetzungen
- Windows-Betriebssystem
- Administratorrechte zur Ausführung des Programms
